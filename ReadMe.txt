This explains prerequisites required to run this package and necessary step to perform assignments.


The following packages must be installed (using command "pip install packagename"):

Pycharm (IDE)
python3.6
robotframework	
robotframework-python3
robotframework-seleniumlibrary
python-dateutil	
python-guerrillamail
selenium
pygments



Packages:

assignment_1.py ( run command -> python assignment_1.py)
assignment_2.robot (run command ->  robot assignment_2.robot)


Note: to run python package we need chrome driver. The path of the chrome driver should be speceifed wither in system path or in the script as mentioned
